How to Post to the Blog

node scripts/new-post.cjs --title "MatBoss vs GymMaster: Which System Actually Increases Martial Arts Enrollments?" --category market --date 2024-12-19



Automated route for Make.com

  1. Rotate the ingestion key:

     powershell -ExecutionPolicy Bypass -File .\scripts\rotate-news-api-key.ps1 -IssuedTo "Make.com"

  2. Copy the printed raw key into the Make.com HTTP module header:

     x-api-key: <printed key>

  3. Send JSON to:

     POST https://matboss.online/api/news-posts

  4. Required JSON fields:

     id, slug, category, title, headline, excerpt, date, thumbnailUrl, readTime, content

  5. Allowed categories:

     product, announcements, policy, company, social, market

Notes

  - thumbnailUrl accepts either a full URL (https://...) or a local site path (/news/thumbnails/example.svg).
  - content can be an array of strings. A plain string is also accepted and split into paragraphs.
  - Posts are stored in the blog database and merged with the existing static seed posts on /news.

Manual fallback for local seed content
